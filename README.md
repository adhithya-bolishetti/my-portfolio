# Adhithya Bolishetti - Portfolio Website

A modern, responsive portfolio website showcasing my skills, projects, education, and experience as an aspiring software developer.

## Live Demo

[Click here to view the live portfolio](https://my-portfolio-seven-puce-39.vercel.app/)  

## Features

### Core Features
- **Fully Responsive Design**: Works seamlessly on mobile, tablet, and desktop
- **Six Professional Sections**: 
  - Hero/Introduction with contact details
  - Education timeline with achievements
  - Project showcase with GitHub links
  - Internship experience
  - Technical skills categorization
  - Secure contact with resume request system
- **Modern UI/UX**: Clean design with smooth animations and transitions

### Interactive Elements
- Smooth scrolling navigation with active states
- Mobile-friendly hamburger menu
- Animated skill cards with hover effects
- Project cards with technology tags
- Resume request form (no direct download)
- Form validation and user feedback
